15  * @file memoryManagement.h
16  * The main properties of this algorithm is:
17  * The total memory is divided in to two major part one part is used to manage
18  the data structure and other part is used for operation.A very less memory
19  has been used to manage the data.
20  The major part is further divided in to 10 blocks-
21  Blocks of 8 byte,16 byte,32 byte,64 byte,128 byte,256 byte,512 byte,
22  1024 byte,2048 byte and Last block is of not specified byte it is defined
23  as other area.
24  For memory <= 8 byte memory is allocated from blocks of 8 byte.
25  For memory <= 16 byte memory is allocated from blocks of 16 byte.
26  and so on,if memory is greater than 2048 byte it will we given from other area.
27  * SIZE can be changed and full memory architecture will be defined according to
28  the given SIZE.The percentage of the size of blocks of 8,16,32...2048 bytes will
29  be same,The number of blocks depend on SIZE.
30  * one bit space are used for maintaining the status of blocks,to show whether it
31  is filled or not.
32  * Less than 1% of total memory is used for maintaing the status.
33  * Every calculation are done using MACRO.
34  * one more function is defined to dispaly the allocated and free memory.
35  * Initially the SIZE is defined as 1 MB and We can change the SIZE according to
36  users need.
37  *This program has the library functions for allocating memory,
38  deallocating memory and displaying the size of free memory and
39  allocated memory.
40 
41  *memory_alloc function - This function provides memory dynamically to
42  user.
43 
44  *memory_free function - This function frees the space allocated for the
45  user in the created heap space under user's
46  request.
47 
48  *memory_calloc function - This function provides memory dynamically to
49  user in asked number of blocks of specified
50  size and intialise them to zero.
51 
52  *memory_realloc function - This function reallocates memory to the
53  previously allocated memory pointer by asking
54  the user ,the already allocated pointer and
55  size to be allocated.
